    Quick Start Rules¶

    This is a summary of the Q* RPG System. Explanations are shorter and more to the point, but the the descriptions aren’t as detailed and edge cases may be disregarded altogether. If something is unclear, look up the full rules!

    You play a character, and the GM manages non-player characters, the environment, and tells the story. Every character is defined by a set of attributes. Attributes include abilities, skills, means, equipment, experience, race, perks, flaws, etc.
